FLASK_APP Environment Variable Set:
Flask needs to know which file contains your application instance.

Temporary Setting (for current session):
# For PowerShell on Windows
$Env:FLASK_APP = "app.py"

Permanent Setting:
You can add the above line to your virtual environment's activation script 
(Activate.ps1) to set it automatically each time you activate the environment.
